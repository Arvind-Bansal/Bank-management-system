# Bank Management System

A Python-based console application that simulates core banking operations such as account management and transaction handling. This project demonstrates the use of file handling, modular programming, and basic system design principles.

## 🚀 Features

* Create a new bank account
* Deposit money into an account
* Withdraw money with balance validation
* Check account balance
* Store and retrieve account data using file handling

## 🛠️ Tech Stack

* Python
* File Handling (for persistent data storage)
* Json

## 📂 System Design

The application follows a modular structure where each banking operation is handled by separate functions.
User data is stored in files, allowing persistence across multiple runs of the program.

## ▶️ How to Run

1. Clone the repository
2. Navigate to the project directory
3. Run the Python file:

```bash
python main.py
```

## 📌 Use Case

This project simulates a simplified banking system and helps in understanding how real-world systems manage user data and transactions at a basic level.

## ⚠️ Limitations

* Console-based application (no graphical interface)
* Data stored in files instead of a database
* No authentication or security mechanisms
* Not suitable for real-world banking use
